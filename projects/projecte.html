Project E

Project Type: University Project
Software Used: Unity3D
Languages Used: C#
Roles: Programmer

About

Project E is a 3rd person adventure and exploration game with procedural generated worlds where the player is placed in a world of misteries and tries to discover hidden secrets left by an extinguished civilization. The project is being developed by a group of 5 people and this first demo was made as a final project for the course of Video Game Design in IPB college.

Roles

Programmer

As the only Programmer, I was responsible for making all the scripts for the game. I implemented the procedural terrain generation tool used to generate the game terrain, all the gameplay mechanics, the enemy AI, the UI and much more.
