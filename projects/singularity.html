Singularity

Project Type: University Project
Software Used: Unity3D
Languages Used: C#
Roles: Programmer

About

Singularity is an Arcade game heavily inspired by classics like Arkanoid and Breakout where the player controls a spaceship that tries to defend the world against hordes of invading AI controled robots. The project was made for an "interdisciplinary week" hosted in IPB college which the theme was "Future" and it was developed in 1 week.

Roles

Programmer

As the only programmer in the team, I was responsible to implement all game mechanics like the spaceship movement, Power Ups and ball physics. I also made the VFX for the game as well as a tool for level creation that let the Level Designer create levels in a txt file with ones and zeroes.
