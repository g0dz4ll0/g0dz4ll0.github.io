Project Survival

Project Type: Personal Project
Software Used: Unity3D
Languages Used: C#
Roles: Solo Dev

About

Project Survival is a a personal hobby project that consists of a survival game with procedural generated levels inspired by Valheim with a cartoony style;. The project is being developed solely by me.

Roles

Solo Dev

As a solo dev, I am responsible for making all the contents of the game including all mechanics implementation, UI/UX, design, 3D models, shaders, sounds and VFX. As of right now, most visual assets are placeholders reused from older projects or from the Unity Asset store.
