All Natural

Project Type: University Project
Software Used: Unity2D
Languages Used: C#
Roles: Programmer

About

All Natural is a 2D game about a baby whose trying to be healthy and avoid plastic food. Try to resist waves of various foods and avoid the bad ones while trying to eat healthy and beat your highscore! The game was made in 1 day for a Game Jam hosted in IPB college where the theme was "All Natural".

Roles

Programmer

As the only programmer in the team, I was responsible to implement all game's mechanics such as player movement and the wave system. I also implemented all the UI elements as well as its functionality and the sprites for the food and players.
